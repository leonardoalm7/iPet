# iPet: Ecossistema Digital para Bem-Estar e Logística Pet 🐾

O **iPet** é uma plataforma Super App projetada para centralizar a jornada de cuidados de animais de estimação, conectando responsáveis, clínicas e grandes players do setor aéreo através de tecnologia **Blockchain** e **IoT**.

## 🚀 Visão Estratégica do Ecossistema

Nossa arquitetura foca na resolução de dores burocráticas para gerar engajamento e recorrência:

- **iPet Pass (Produto de Entrada)**: Passaporte digital imutável focado em **Compliance Sanitário as a Service (CSaaS)**.
- **Saúde Inteligente**: Carteira de vacinação digital com alertas proativos e agenda automática.
- **Rede Integrada**: Localização de clínicas e marketplace com programa de loyalty.

## 📂 Organização do Repositório

A estrutura foi desenhada para suportar a escala do projeto e facilitar a colaboração técnica:

- **`/docs/business`**: Documentação estratégica, incluindo o Business Case completo do Smart Pet Pass e metas operacionais (como a redução no tempo de check-in).
- **`/docs/architecture`**: Modelagem de domínio (**DDD**) e definição da Linguagem Ubíqua do ecossistema.
- **`/docs/requirements`**: Especificações técnicas e User Stories detalhadas para o desenvolvimento do backend.
- **`/prototipos`**: Módulos funcionais em desenvolvimento, começando pelo **Agente de Aeroporto** (Next.js).

---

## 🛠️ Como Contribuir (Time Técnico)

1. Certifique-se de ter o Node.js instalado.
2. Navegue até o protótipo desejado dentro de `/prototipos`.
3. Execute `npm install` para baixar as dependências locais (não versionadas no Git).
4. Consulte a documentação técnica em `/docs/requirements` antes de iniciar novas features.
